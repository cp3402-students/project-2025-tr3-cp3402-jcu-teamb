# Baizonn Learning Centre Website Project

This project involves the design, development, and deployment of a custom WordPress website for **Baizonn Learning Centre**. The goal is to increase student enrolment by showcasing the centre's unique **F.A.S.T.** methodology, S.T.E.M curriculum, and experienced teaching staff.

The site is built using a custom WordPress theme developed from scratch using the `_s` (Underscores) starter theme.

*   **Custom Page Template:** Automated "Our Team" page using a custom PHP loop.
*   **Responsive Design:** Optimized for Desktop, Tablet, and Mobile.
*   **Dynamic Content:** Teachers and Events are managed via WordPress Posts.
*   **Performance:** Minimalist CSS and optimized assets.
